Gaston: Julia plotting using gnuplot
====================================

    using Gaston
    
    gaston_demos()
    gaston_tests()

Installation
------------

Gaston requires gnuplot to be installed in your system. It has been tested
With version 4.6.

To install Gaston, run

    julia> Pkg.add("Gaston")
